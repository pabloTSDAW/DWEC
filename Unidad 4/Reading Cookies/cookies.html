What is a cookie?

Las cookies son simples archivos de texto que un servidor web almacena en la máquina de un usuario. Este archivo de texto contiene información en un par nombre-valor, que puede ser recuperado por un sitio web. Este valor único se usa para distinguir entre múltiples usuarios que están accediendo al sistema. Una cookie solo puede ser recuperada por el sitio web que emitió esa cookie.

Cookies are simple text files that are stored in a user’s machine by a web server. This text file contains information in a name-value pair, which can be retrieved by a website. This unique value is used to distinguish between multiple users who are accessing the system. A cookie can only be retrieved by the website who issued that cookie.


What cookies are used for?

  • Almacenar la información de estado actual de un sitio web en la computadora del usuario
  • Un sitio web puede identificar de manera única el número de visitantes. Las cookies también se pueden utilizar para categorizar a los visitantes repetidos y pueden contar la cantidad de veces que un usuario particular visita el sitio web.
  • Almacenar preferencias y configuraciones, y estos datos se pueden usar para recordar las preferencias del usuario y, por lo tanto, proporcionar una personalización única para usuarios particulares.
  • Para rastrear el tiempo que un usuario particular pasa en el sitio web y para encontrar los enlaces en los que el usuario hace clic en ese sitio web en particular.
  • Store the current status information of a website on the user's computer
  • A website can uniquely identify the number of visitors. Cookies can also be used to categorize repeated visitors and can count the number of times a particular user visits the website.
  • Store preferences and settings, and this data can be used to change user preferences and thus provide unique personalization for particular users.
  • For the time that a particular user passes on the website and to find the links on which the user clicks on that particular website.

How Cookies Are Initialized in Browsers?

Cuando escribimos una URL, el navegador envía una solicitud al sitio web para la página. Al hacer esto, el navegador también busca una cookie en la computadora del usuario. Si el navegador encuentra una cookie para ese sitio web en particular, enviará ese par de nombre-valor a ese servidor web asociado con la URL. El servidor recibe el par nombre-valor y la solicitud de la página y utiliza los datos de la cookie para producir la experiencia de usuario única al proporcionar preferencias particulares y esquemas de color.

When we type a URL, the browser sends a request to the website for the page. On doing this, the browser also looks for a cookie in the user’s computer. If the browser finds a cookie for that particular website, it will send that name-value pair to that web server associated with the URL. The server receives the name-value pair and the request for the page and uses the cookie data to produce the unique user experience by providing particular preferences and color schemes.


What types of Cookies do you know?

  • Cookies transitorias

    • Este tipo de cookie se almacena en la memoria RAM y contiene una ID de sesión que almacena información sobre la sesión de navegación del usuario. Se eliminan de la computadora tan pronto como el navegador se apaga.

  • Cookies persistentes

    • Este tipo de cookies se almacenan en el disco duro de la computadora y contienen información sobre las preferencias del usuario para un sitio web. Estas preferencias se pueden usar para sesiones de exploración adicionales y se conservan mientras el usuario las permita. Se almacenan permanentemente y no se eliminan cuando se cierra el navegador.

  • Flash cookies

    • son pequeños archivos Flash, como videoclips o gifs almacenados en la computadora de un usuario por sitios web. No se pueden eliminar mediante ningún mecanismo desde el nivel del navegador. Solo se pueden eliminar manualmente o mediante algunos complementos especiales.


  • Transient cookies

    • This type of cookie is stored in RAM and contains a session ID that stores information about the user's browsing session. They are removed from the computer as soon as the browser powers down.

  • Persistent Cookies

    • This type of cookies are stored on the computer's hard drive and contain information about the user's preferences for a website. These preferences can be used for additional browsing sessions and are preserved as long as the       allow them. They are permanently stored and are not deleted when the browser is closed.

  • Flash cookies

    • They are small Flash files, such as video clips or gifs stored on a user's computer for websites. They can not be removed by any mechanism from the browser level. They can only be deleted manually or with some add-ons       special.


What are the common risks of cookies?

  • Ataque de falsificación de solicitud entre sitios (XSRF)
  • Fijación de sesión
  • Creación de scripts entre sitios
  • Ataque de lanzamiento de cookies
  • Ataque de desbordamiento de cookies

  • Cross-site Request Forgery Attack (XSRF)
  • Session Setting
  • Cross-Site Scripting
  • Cookie Launch Attack
  • Cookie overflow attack
